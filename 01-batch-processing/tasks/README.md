# Batch Processing - Tasks

This directory contains all batch processing tasks (13 total).

## Tasks Overview

1. **task-01-read-structured-data** - Read structured and semi-structured data
2. **task-02-simple-batch-app** - Simple batch processing application
3. **task-03-execute-jobs** - Execute batch processing jobs
4. **task-04-data-collections** - Work with data collections in batch processing
5. **task-05-error-handling** - Handle errors
6. **task-06-various-sources** - Read and parse data from various data sources
7. **task-07-system-components** - Execute jobs and demonstrate system component relations
8. **task-08-optimize-performance** - Use transformations and actions to optimize performance
9. **task-09-incremental-loading** - Implement incremental data loading
10. **task-10-complex-files** - Read and parse data from complex files
11. **task-11-monitoring-logs** - Execute and debug jobs through monitoring and logs
12. **task-12-performance-optimization** - Performance optimization techniques
13. **task-13-slowly-changing-dimension** - Implement slowly changing dimension
