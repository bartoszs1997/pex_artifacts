# 11 — Transpose a Dataset from Rows to Columns and Vice Versa

## Goal

Demonstrate PIVOT and UNPIVOT operators to transpose table data between row and column formats.

---

## Concepts used

| Category | Operator |
|---|---|
| Rows to columns | `PIVOT` with aggregation |
| Columns to rows | `UNPIVOT` |

---

## What the script does

| Section | What it demonstrates |
|---------|----------------------|
| 1 | `PIVOT` — quarterly revenue rows become Q1, Q2, Q3 columns per person |
| 2 | `UNPIVOT` — Q1, Q2, Q3 columns transformed back into rows |

| PIVOT | Section 1 |
